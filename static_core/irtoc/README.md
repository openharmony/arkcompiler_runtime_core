# irtoc

irtoc is the Ruby DSL plus backend pipeline that generates interpreter handlers, fastpaths, `NativePlus` entrypoints,
and boundary stubs for `static_core`.

Start with:

- `AGENTS.md` for task routing, validation layers, and debugging hints
- `../docs/irtoc.md` for the current build pipeline, modes, artifacts, and usage guidance
- `../docs/irtoc-intrinsics-for-interpreter.md` for interpreter-facing `FastPathPlus` / `NativePlus` behavior
